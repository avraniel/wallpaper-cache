## ✨ Features

- 🎨 **Beautiful GUI** - Built with CustomTkinter for a modern dark theme
- 🔍 **Multi-Source Search** - Wallhaven, Reddit, and MoeWalls support
- 📥 **Smart Download Queue** - Persistent queue with retry logic
- 🗂️ **Collections** - Save and manage wallpaper collections
- 🔑 **API Key Management** - Encrypted storage for API keys
- 📊 **History Tracking** - Keep track of all downloads
- 💾 **Disk Management** - Automatic cleanup with size limits
- 🚀 **Startup Downloads** - Auto-download on application start
- 🐟 **Fish Shell Integration** - Convenient abbreviations
- 🖥️ **Desktop Integration** - Autostart support for Niri and KDE
- 🌐 **Multi-Page Search** - Search across multiple pages
